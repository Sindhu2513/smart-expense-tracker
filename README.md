## ğŸš€ Getting Started

### Prerequisites

### Installation

Clone the repository:

```bash
git clone https://github.com/Sindhu2513/smart-expense-tracker.git
```

Navigate to the project directory:

```bash
cd smart-expense-tracker
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

Application runs at:

```
http://localhost:3000
```
